# 输入输出指令
## 输入指令
IN DST,SRC
源操作数是一个端口地址
* 若分配端口地址在0~255，可以采用直接寻址(直接写出端口地址)
* 若分配端口地址超出255，必须采用间接寻址(范围0~65535)
```nasm
MOV DX,端口地址
IN AL,DX;注意不用打中括号，且必须位DX，字操作用AX，字节操作用AL
```
## 输出指令
OUT DST,SRC
* 若分配端口地址在0~255，可以采用直接寻址(直接写出端口地址)
* 若分配端口地址超出255，必须采用间接寻址(范围0~65535)
